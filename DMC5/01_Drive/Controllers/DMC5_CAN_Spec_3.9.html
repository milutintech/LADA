Some CAN Bus information

All Messages are listed as transmit messages. The column 'node name' describes the sender of the message.

The bus Speed is 500k.

The table below shows the bit numbering for motorola formated bit fields. Each row represents one byte (from Byte 0 to Byte 7) The Byte index is on the right side.
The green signal would be markes with startbit=12 and a bit size of 12. The most significant bit is on bit 12 and the least significant bit on bit position 23.
